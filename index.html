About the Website

Crochet is a process of creating textiles by using a crochet hook to interlock loops of yarn, thread, or strands of other materials. The name is derived from the French term crochet, meaning 'small hook'.

If you are interested in picking up this skill, this website is for you! Draw inspiration from some of my crochet pieces, learn all the different crocheting techniques, and see patterns to start creating your own crochet piece!
